## Projects

### 🔐 Self-Hosted IAM Lab
A fully containerized identity stack running **Keycloak**, **OpenLDAP**, and **Jenkins** via Docker Compose. Covers OIDC/SSO flows, LDAP user provisioning across HR/IT/Accounts OUs, group-to-role mapping, and HTTPS with a custom CA.

**Tech:** Keycloak · OpenLDAP · Jenkins · Docker Compose · OIDC · LDAP

---

### 🌐 IAM Portfolio Website
A full-stack portfolio site built with Next.js (Vercel), Node.js/Express (Render), and PostgreSQL (Supabase) to document and display my IAM journey publicly.

**Tech:** Next.js · Node.js · PostgreSQL · Supabase · Vercel · Render
